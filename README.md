###### 	低三层:
- 物理层主要作用是定义物理设备如何传输数据
- 数据链路层在通信的实体间建立数据链路连接
- 网络层是为数据在结点之间传输创建逻辑链路

------------


###### 	传输层
- 向用户提供可靠的端到端（End to End）服务
- 传输层向下层屏蔽了下层数据通信的细节
